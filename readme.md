# The Mini Portfolio - Jheny Chan

This is my first responsive web project for Week 4 of the Mapping Program.

## Project Description
A simple profile card built with a **Mobile-First** approach. The design adapts to different screen sizes using CSS Media Queries.

## Tech Stack
* **HTML5**: For the structure.
* **CSS3**: For styling and responsiveness.

## Why These Colors?
I chose **Peach/Coral** because it represents the strawberry theme of my profile picture. It feels fresh and warm!

## How to View
1. Clone this repository.
2. Open `index.html` in your browser.
