# 🍓 The Mini Portfolio - Jheny Chan

Hello! This is my project for **Week 4** of the Mapping Program.

## 📱 Features
* **Mobile-First Design**: The layout is optimized for small screens first.
* **Responsive Layout**: The background color changes when viewed on a desktop (768px or wider).
* **Cute Aesthetic**: Using a strawberry theme!

## 🛠️ Tech Stack
* HTML5
* CSS3 (Flexbox & Media Queries)

## 📝 Learning Journey
In this project, I learned how to use **Media Queries** to make my website look good on both phones and laptops. I also learned how to use **GitHub** for the first time!
